# cloudreve 是国内大佬 Aaroon 开发的一个开源网盘项目


官网：cloudreve.org
github：https://github.com/cloudreve/cloudreve

# 首先我们打开网盘左测的 WebDAV

  • 创建一个新账号（记住账号名和密码待会用到）

# 在 windows 下打开注册表

  1. 路径
    计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WebClient\Parameter
  2. 把 BasicAuthLevel 值改成 2
  3. 删除目录下的 FileSizeLimitInBytes
  4. 在当前目录新建一个 FileSizeLimitInBytes
  5. 右键修改数值
    十六进制 ffffffffffffffff
    十进制 18446744073709551615

# 打开服务

  1. 找到 WebClient
  2. 设为自动，然后停止再启动

# 打开此电脑，在左上计算机中选择映射网络驱动器

  1. 在文件夹中输入地址
  2. 之后弹出输入账号和密码，复制网站上的账号密码输入上去
  3. 至此成功添加到此电脑中，来看看效果

  • 感谢各位网上的大佬帮助以及 cloudreve 官方提供这么棒的开源网盘
