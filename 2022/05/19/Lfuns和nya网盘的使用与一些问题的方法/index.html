# Lfuns 和 nya 网盘的使用与一些问题的方法

  • 首先说明 Lfuns 是我自建的网盘，和 nya 网盘基本没区别，使用上喜欢即可。
  • nya 网盘：nyapan.mouup.top
    Lfuns：www.lolifuns.top

# 使用方法

# 视频外链

  1. 首先要当然要上传视频或图片啦，轻按下方加号即可上传（新建目录可以新建文件夹）
  1. 上传完视频后右键文件，跳出获取外链
  1. 把外链复制
  1. 投稿喵站的话，直接在视频上传上放置链接即可

# 图片外链（这里介绍我自己使用的方法，如果有更好的方法有可以在评论里说）

  1. 进入一个 markdown 的编辑器（这里用菜鸟工具演示，在线 markdown 编辑器）
  2. 复制图片外链
  1. 在左边输入 ! [] (图片外链)
  1. 选中图片 Ctrl+c，然后打开喵御宅，发帖界面 Ctrl+v 即可

# 关于 nya 网盘遇到的一些问题和解决方法

# 登入问题

  • 不小心登入退了出去，在一直刷新页面，如果有在浏览器保存账号密码，浏览器也会一直弹出登陆密码，可以尝试拼手速进去（
  • 也可以尝试换一个浏览器登入，或者清除浏览器的缓存或者 cookie（清除 cookie 不是很建议）

# 文件上传了，看不到文件

  • 请在右上角的排列方式切换即可显示。

  • 当然也有人说没有这个图标，可以尝试刷新网页，图标就会出现（当然也有可能很快就消失）
