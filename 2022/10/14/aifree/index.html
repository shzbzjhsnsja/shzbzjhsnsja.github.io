# 资源包下载

https://www.lolifuns.top/s/7BhO (密码：1234)

# 一、前往 GitHub 下载 ZIP 压缩包，解压出来

https://github.com/yuyuyzl/EasyVtuber

# 二、点击 01A. 构建运行环境

# 三、然后点击 00B. 预训练数据下载进入网站下载训练数据（整合包里包含）

解压到 EasyVtuber-main\data\models 文件夹中

# 四、返回 EasyVtuber-main 点击 02B. 启动器（调试输出）

# 五、使用 ai 生成一张喜欢的形象，ps 抠图

  • 完成抠图后，ps 打开 EasyVtuber-main\data\images 中的 lambda_00.png
  • 使之与 lambda_00.png 的脸部重合，然后隐藏 lambda_00.png 图层，保存在 EasyVtuber-main\data\images

# 六、安装 stream FX

本人使用的 stream FX 0.11.1 和 obs 27.1.3
安装时注意版本匹配，stream FX 暂不支持最新版 obs
https://github.com/Xaymar/obs-StreamFX

# 七、下载着色器文件

https://github.com/shugen002/shader/blob/master/merge alpha2.hlsl

# 八、打开 obs

  1. 新建场景
  2. 然后再来源上添加视频采集设备
  3. 返回文件中点击 02B. 启动器开启模型
  4. 选择 Output 中的 obs，和 Face Date Source 中的 Mouse Input 测试

# 九、在来源中的视频采集器右键滤镜

  • 然后点击左下角的 +
  • 然后添加着色器，选择刚刚下载好的着色器文件

# 十、使用 ios 面部，或者 pc 摄像头。


在 Face Date Source 中可以选择

  1. ios 的可以使用 iFacialMocap，用 ios 的面部效果会好些
    https://apps.apple.com/us/app/ifacialmocap/id1489470545?mt=8
  2. 第二个 webcam 就是 pc 面部啦，但效果就是比较差
    你打开 OBS 点开随便一个视频源的属性，在设备下拉列表里找到你要用的电脑摄像头，数一下他是这个下拉菜单的第几个然后在代码 main.py 的第 461 行找到 cap = 专栏行为准则 .VideoCapture (0)，把这一行的 0 改成你摄像头的第几个减一，比如菜单第二个是你的摄像头你就改成 1，然后保存，在启动器里选 Webcam 就可以用了
  3. 第三个就是鼠标输入了，会跟着鼠标指针动
