# 本地 NovelAI 配置教程

# 配置推荐

4GB 以上显存的显卡（N 卡）
推荐：3060 12G 显存
CPU 无所谓 内存 16G 或以上

# 安装前的环境准备

需要 cuda、python3.10、git
cuda 下载地址：推荐 11.3 https://developer.nvidia.com/cuda-11-3-1-download-archive
cudnn 下载：https://developer.nvidia.com/rdp/cudnn-archive（需要 nVidia 开发者账号）
python 下载地址：https://www.python.org/downloads/
git 下载地址：https://git-scm.com/download/win

# 新建一文件夹拉取项目

右键 git bash here 进入命令行输入：git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git 拉取项目

# 然后放入模型

  • 打开 models 文件夹再打开 Stable-diffusion 放入我们所需的模型（两个文件名需一致）
  • 退回 models 文件夹，创建一个名为 hypernetworks 的文件夹，进入里面把 pt 文件放进去

# 返回 stable-diffusion-webui 文件夹

点击 webui-user.bat 运行命令行就会自动安装依赖（此步建议全程开启代理）

如果依然无法安装成功依赖可以使用上面的整合包

# 完成后不要关闭命令行，打开浏览器输入 http://127.0.0.1:7860

  • Sampling Steps 建议设置到 40-50 左右，CFG Scale 建议设置到 11.
  • 负面 tag 规避词：lowres, bad anatomy, bad hands, text, error, missing fingers, extra digit, fewer digits, cropped, worst quality, low quality, normal quality, jpeg artifacts, signature, watermark, username, blurry
  • 负面 tag 可以加上 extra limbs,3 legs 规避一些猎奇发生
  • 正面 tag：可以加上 masterpiece, best quality 增强效果
  • 一些 tag 的参考表 https://www.lolifuns.top/s/QOIL 密码：1234
